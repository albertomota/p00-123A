Any patches to contrib modules should go in this directory.
Pleae make sure that the patch filename includes the name
of the module it applies to, the node ID from drupal.org that
the patch was posted to, and the comment ID of the actual
comment it was posted to.  This will make tracking whether
the patch has been commited to the effected module easier.

EXAMPLE:

views-has_aggregate-1512488-10.patch
